For our CSE165 final project, we will be making a multi-player laser tag shooter game.

These are the features we are now implementing for 7 points:

  1. Multi-User - 1 vs 1 (2 points)
  2. Spatial Audio - laser shots and hit sounds (1 point)
  3. Symbolic Input - custom nameplates with virtual pen (2 point)
  4. Wayfinding - show objective location (1 point)
  5. System Control - change laser color (1 point)
